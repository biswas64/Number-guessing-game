A console-based number guessing game with 5 difficulty modes.

## Modes
- **Easy**: 100 chances (basically unlimited)
- **Normal**: 7 chances
- **Hard**: 3 chances
- **Brutal**: 1 chance only (one mistake and you lose)
- **Idiot**: Just a mode for idiots

## Features
- Smart graduated hints (tells you how close you are)
- Robust input validation
- Play again option
- Reveals answer on loss
